Sketches

Each app has Tah sketches that go with it.

To upload a sketch to Tah, follow the steps given below :

Install Arduino

You can upload sketches to the Tah using the Arduino IDE. It is open source and can be used with Mac, Linux and Windows. If you follow the instructions on the above links, you will have a green LED blinking on your Tah within minutes.

For any troubleshooting, you can use our forum.

Download sketches from GitHub

  1. You can download the latest source code for our Tah sketches from GitHub here.
  2. Rename the downloaded archive from sketches-master to Tah.

Install Tah Library

  1. Do not unzip the downloaded library, leave it as is. In the Arduino IDE, navigate to Sketch > Import Library. At the top of the drop down list, select the option to "Add Library".
  2. You will be prompted to select the library you would like to add. Navigate to the .zip file's location and open it.
  3. Return to the Sketch > Import Library menu. You should now see the library at the bottom of the drop-down menu. It is ready to be used in your sketch.

For a more complete guide on installing a library, check out Arduino's guide.

Upload the default sketch

  1. Connect your Tah board to your USB port.
  2. Select File->Examples->Tah->ArdSCL
  3. Compile and Upload the sketch

Check out the start page

Follow the guidelines given on the Getting Started page for Tah to blink a LED using your smartphone.

You can check out the examples page to know more about each sketch in detail.
